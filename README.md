# SampleARCoreViews
Learn to render simple Views in the scene and create models programmatically with Android ARCore.

## Requirements
* Android Studio 3.5 Canary 6
* Kotlin 1.3.30-eap-11
* Android Gradle Plugin 3.5.0-alpha06
* Gradle wrapper 5.2.1
* AAPT 2
